# AgentHire - AI Hiring Agent

## Problem

Recruiters spend significant time manually screening resumes, matching candidates to job descriptions, and preparing interview questions.

## Solution

AgentHire is an AI-powered recruitment workflow that automates candidate evaluation through multiple AI agents.

## Workflow

1. Resume Upload
2. Resume Parsing
3. Skill Extraction
4. Candidate Matching
5. Recommendation Generation
6. Interview Question Generation
7. JSON Report Creation

## Tech Stack

* Python
* PyPDF
* GitHub
* Google Gemini (planned)
* UiPath (planned)
